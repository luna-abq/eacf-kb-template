# 2. Departments

Per-department context. Each department gets its own subfolder.

## Structure

```
2-departments/
  1-engineering/
    README.md
    1-overview.md
    2-processes.md
  2-marketing/
    README.md
  ...
```

## What Goes Here

- Team responsibilities and ownership
- Repeatable processes and SOPs
- Onboarding guides
- Inter-team interfaces

## Start Here

Create a subfolder for your first department: `2-departments/1-[department-name]/README.md`
